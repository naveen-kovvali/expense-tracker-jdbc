# Expense Tracker (JDBC Project)

## 📌 Description
A console-based Java application to manage daily expenses using JDBC and MySQL.

## 🚀 Features
- Add expenses
- View all expenses
- Update expenses
- Delete expenses
- Total expense calculation
- Category-wise report

## 🛠 Tech Stack
- Java
- JDBC
- MySQL

## 🧠 Concepts Used
- JDBC API
- PreparedStatement
- SQL (CRUD + GROUP BY)
- Exception Handling

## ▶️ How to Run
1. Clone the repository
2. Import into VS Code / any IDE
3. Create database and table
4. Run MainApp.java

## 📂 Project Structure
- DBConnection.java
- AddExpenses.java
- ViewExpenses.java
- UpdateExpense.java
- DeleteExpense.java
- ReportExpenses.java
- MainApp.java
